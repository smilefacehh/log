## 特性
在robolog的基础上增加以下功能
1. 支持不同的logger自定义格式 ok
2. 支持不同的logger自定义等级（低优先级）
3. 支持console颜色区分头部、内容 ok
4. 支持插入外部实现，将日志打印到同一个文件或console里面 ok
5. 支持按大小切割文件（低优先级）
6. 支持每次运行重新写一个新的日志文件，不切割，主要用于调试 ok
7. 支持每次运行重新写一个新的日志文件，按文件大小切割
